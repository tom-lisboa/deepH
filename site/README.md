# deepH Docs Site (Next.js)

Landing page + documentação do `deepH` em Next.js (`site/`).

## Rodando localmente

```bash
cd site
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Rotas

- `/` landing page
- `/docs` documentação completa
- `/docs/hello-worlds` catálogo interativo de Hello Worlds (linguagens + Tailwind + shadcn/ui)
- `/docs/calculadora` tutorial step-by-step (calculadora no deepH)
- `/docs/comparativo-claude-code` comparativo prático com Claude Code
